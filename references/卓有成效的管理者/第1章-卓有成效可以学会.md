# 第一章：卓有成效可以学会

## 核心观点

### 一、基本观念

> **"对组织负有责任，能影响组织经营成果的人，就是管理者。"**

> **"管理者，就必须卓有成效。"**

> **"卓有成效是一种习惯，是可以学会的。"**

### 二、管理者的范围

管理者不限于高层管理人员。所有负责行动和决策而又有助于提高机构工作效能的人，都应该像管理者一样工作和思考。

**知识工作者、经理人员和专业人员**，不论其是否有下属，不论职位高低，只要因其职位和知识必须做出决策，都应该被视为管理者。

### 三、卓有成效的特征

一位卓有成效的管理者，一般具有以下**六个特征**：

| 特征 | 说明 |
|-----|------|
| 1. 重视目标和绩效 | 只做正确的事情 |
| 2. 一次只做一件事 | 只做最重要的事情，审慎设定优先顺序 |
| 3. 知道自己所能做出的贡献 | 创造新思想、远景和理念；原则是"我能做哪些贡献？" |
| 4. 注重出色的绩效和正直的品格 | 在选用高层管理者时尤为如此 |
| 5. 增进沟通的重要性 | 有选择性地搜集所需要的信息 |
| 6. 只做有效的决策 | — |

---

## 卓有成效的五项习惯（核心）

### 习惯一：管理时间

**有效的管理者知道他们的时间用在什么地方。**

- 他们所能控制的时间特别有限
- 他们会系统地工作，来善用这些时间
- 时间管理是卓有成效的基础

**时间管理的四个步骤：**

| 步骤 | 说明 |
|-----|------|
| 1. 界定任务 | 时间无法管理的东西，可以用任务去管理 |
| 2. 记录时间 | 连续坚持四周，检查是否将70%-80%的时间用于完成工作任务 |
| 3. 管理时间 | 消除浪费时间的活动 |
| 4. 统一安排时间 | 将可自由运用的时间化零为整成连续性时间 |

**消除浪费时间的四种情况：**

| 情况 | 说明 | 解决方式 |
|-----|------|---------|
| 周期性的危机 | 如冬天仓库着火 | 将例外管理变成例行管理 |
| 人员过剩 | 太多时间用于处理人际关系 | 减少不必要的人际协调 |
| 组织不健全 | 会议太多 | 优化会议效率 |
| 信息不畅 | 需要的信息得不到 | 建立有效的信息系统 |

> **"时间没有替代品。时间无法储存。时间没有弹性。"**

### 习惯二：重视贡献

**有效的管理者重视对外界的贡献。**

- 他们并非为工作而工作，而为成果而工作
- 不会一接到工作就一头钻进去，更不会一开头就探究工作的技术和手段
- 首先自问："别人期望我做出什么成果？"

**贡献的三层含义：**

| 层次 | 说明 |
|-----|------|
| 直接成果 | 看得见的业绩贡献 |
| 价值实现 | 建立和坚持的价值观 |
| 未来人才发展 | 培养明日的人才 |

**贡献的两个维度：**

| 维度 | 说明 |
|-----|------|
| 向上看、向外看 | 关注组织外部的成果 |
| 发挥经理人的贡献 | 从"我想做什么"转变为"我应该做什么" |

> **"重视贡献是有效性的关键。所谓有效性，就是使能力和知识资源能够产生更大成果的一种手段。"**

### 习惯三：善于利用长处

**有效的管理者用人所长，包括自己的长处、上司的、同事的、下属的。**

**核心原则：**
- 善于抓住有利形势，做自己想做的事
- 不会把工作建立在短处上，也不会做自己做不了的事

**四个用人维度：**

| 维度 | 说明 |
|-----|------|
| 利用自己的长处 | 认识并发挥自己的优势 |
| 利用上司的长处 | 支持而非挑战上司 |
| 利用同事的长处 | 协同合作 |
| 利用下属的长处 | 着眼于机会而非问题 |

**考评方式的四个问题：**
1. 列出对某人过去职务和现任职务所期望的贡献
2. 某人的实际绩效记录与这项期望贡献相对照
3. 哪些方面工作他确实做得很好？
4. 哪些方面他可以做得更好？

> **"管理者如何管理上司？答案是：确保上司的优势发挥出来，支持他做他想做的事。"**

### 习惯四：集中精力于少数重要领域

**有效的管理者集中精力于少数重要的领域。**

- 设定优先次序，坚守优先次序
- 要事第一，重要的事先做
- 勇敢抛弃那些过时的任务，或者推迟做那些次要的任务

**两个重要原则：**

| 原则 | 说明 |
|-----|------|
| 重将来而不重过去 | 摆脱成功的过去 |
| 重视机会而非问题 | 聚焦于机会 |

**"要事优先"的核心：**

> **"成功的秘诀在于集中精力于少数重要的领域，而不是被紧急事务所困扰。"**

> **"想集中精力，全神贯注于一项工作，首先要有足够的勇气，要敢于决定真正该作和真正先做的工作。"**

### 习惯五：有效决策

**有效的管理者做有效的决策。**

- 在"不同意见讨论"基础上做出判断
- 知道快速的决策多为错误的决策
- 不可或缺的决策数量并不多，但一定是根本性的决策

**决策的五个要素（详见第六章）：**

| 要素 | 说明 |
|-----|------|
| 1. 了解问题性质 | 经常性问题 vs 偶发问题 |
| 2. 明确边界条件 | 解决问题的最低需要 |
| 3. 正确方案 | 先求正确，再求妥协 |
| 4. 兼顾执行措施 | 决策必须转化为行动 |
| 5. 重视反馈 | 验证决策的正确性和有效性 |

**反面意见的运用（详见第七章）：**

| 原因 | 说明 |
|-----|------|
| 保护决策者 | 避免成为组织的俘虏 |
| 提供备选方案 | 决策者需要的"另一种方案" |
| 激发想象力 | 从不同视角观察和理解问题 |

> **"有效的决策不是从'众口一词'中得来的，好的决策应该以互相冲突的意见为基础。"**

---

## 有效决策的特征

| 特征 | 说明 |
|-----|------|
| 重视目标而非方法 | 要的是决策所产生的影响，而非技巧 |
| 先见解后事实 | 决策不是从搜集事实开始，而是从个人见解开始 |
| 兼顾执行 | 一项决策如果不能付诸行动，就只是良好的意愿 |
| 重视反馈 | 躬亲查看或建立信息系统获得反馈 |

## 德鲁克经典语录

1. **"管理就是最大限度地激发他人的善意。"**
2. **"管理者的任务不是去改变人，而在于运用每一个人的才干。"**
3. **"如果一项决策没有细化成一步一步的行动步骤，只是一种意愿而已。"**
4. **"对上级负责是管理的基本原则。"**
5. **"管理者的工作必须卓有成效，卓有成效是可以学会的。"**

## 我的理解与应用

1. **有效性可以学会**：卓有成效不是天赋，而是可以通过训练养成的习惯
2. **时间是最稀缺资源**：善用时间是把最重要的事做好的前提
3. **贡献导向**：从"我想做什么"转向"我应该做什么"、"我能贡献什么"
4. **扬长避短**：用人所长而非改人之短
5. **聚焦要事**：一次只做一件事，勇敢抛弃过去成功的包袱
6. **决策深思**：好的决策源于不同意见的碰撞，而非一致同意
