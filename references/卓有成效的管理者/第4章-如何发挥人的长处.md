# 第四章：如何发挥人的长处

## 核心观点

### 一、核心原则

> **"有效的管理者用人所长，包括自己的长处、上司的、同事的、下属的。"**

---

## 二、用人所长的四维度

| 维度 | 说明 | 关键行动 |
|-----|------|---------|
| **发挥自己的长处** | 认识并发挥自身优势 | 了解自己的长处在哪里 |
| **发挥上司的长处** | 支持而非挑战上司 | 理解上司的期望和工作方式 |
| **发挥同事的长处** | 协同合作 | 了解同事能做什么 |
| **发挥下属的长处** | 着眼于机会 | 用人所长，而非改人之短 |

---

## 三、如何管理上司

| 方法 | 说明 |
|-----|------|
| 确保上司发挥长处 | 支持上司做他想做的事 |
| 了解上司的工作方式 | 适应而非改变 |
| 提供支持 | 让上司的工作更容易 |
| 有效沟通 | 及时、准确的信息 |

> **"管理者如何管理上司？答案是：确保上司的优势发挥出来，支持他做他想做的事。"**

---

## 四、自我管理

| 要素 | 说明 |
|-----|------|
| 认识自己 | 了解自己的长处 |
| 发现不足 | 承认自己的短处 |
| 弥补短板 | 在重要领域弥补 |
| 发挥优势 | 集中于长处 |

---

## 五、考评方式

### 四个关键问题

| 问题 | 说明 |
|-----|------|
| 1 | 列出对某人过去职务和现任职务所期望的贡献 |
| 2 | 某人的实际绩效记录与这项期望贡献相对照 |
| 3 | 哪些方面工作他确实做得很好？ |
| 4 | 哪些方面他可以做得更好？ |

### 绩效考评原则

| 原则 | 说明 |
|-----|------|
| 着眼于长处 | 这个人能做什么？ |
| 容人之短 | 短处不能改变，但可以避开 |
| 发挥优势 | 让长处产生成果 |

---

## 六、关键洞察

> **"管理者要关注一个人能做什么，而不是他不能做什么。"**

> **"即使是最有能力的人也有弱点。关心的是一个人能做什么，而不是他不能做什么。"**

> **"充分发挥人的长处，是组织存在的唯一目的。"**

---

## 七、容忍人之所短

| 原则 | 说明 |
|-----|------|
| 短处不可改变 | 人的短处大多是固有的 |
| 避开短处 | 让短处不产生影响 |
| 容人之短 | 在非关键领域容忍 |
| 聚焦长处 | 让长处产生价值 |

---

## 八、人事决策原则

| 原则 | 说明 |
|-----|------|
| 把职位设计得合理 | 普通人能胜任的职位 |
| 注重绩效 | 绩效是晋升的标准 |
| 容许差异 | 不同人适合不同职位 |
| 培养人才 | 为未来储备人才 |

---

## 我的理解与应用

1. **扬长避短**：关注能做什么，而非不能做什么
2. **系统思考**：让每个人的长处形成合力
3. **容人之短**：短处无法改变，避开就好
4. **支持上司**：帮助上司成功也是自己的成功
5. **持续反馈**：让绩效考评成为发展的工具
