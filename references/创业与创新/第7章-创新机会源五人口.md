# 第7章：创新机会源五——人口统计特征变化（Demographic Changes）

## 核心观点

### 人口统计数据的要素

| 要素 | 说明 |
|-----|------|
| 人口数量 | 总量和增量 |
| 规模 | 家庭规模、群体规模 |
| 年龄结构 | 人口金字塔 |
| 人口组合 | 职业、教育、收入分布 |
| 就业情况 | 劳动力参与率、就业结构 |
| 受教育状况 | 教育水平分布 |
| 收入情况 | 收入水平、消费能力 |

---

## 核心特征

> **"人口统计数据最为清晰易懂，而且有着能够预测的结果。"**

- 可以**事先预知**（人口数据变化是缓慢的、可预测的）
- 对**购买什么、由谁购买、购买数量**有重大影响

---

## 关键洞察

> **"人口变化是慢变量，但一旦变化，影响深远且持久。"**

---

## 案例

### 德鲁克的人口预测

| 预测内容 | 20世纪70年代，美国大学生数量将超过千万 |
|---------|--------------------------------------|
| 当时反应 | 多数大学不接受这一预判 |
| 行动者 | 佩斯大学提前布局 |
| 结果 | 获得了巨大增长 |
